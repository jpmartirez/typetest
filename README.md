
# Typing Test (Next.js)

-> Simple typing test app built with Next.js

-> User selects a time
-> User starts typing
-> Timer runs
-> WPM result is shown at the end

------------------------
Features
------------------------
* Choose typing duration
* Type the given text
* Auto timer start
* Show WPM score

------------------------
How it works
------------------------
1. Select time
2. Start typing
3. Timer ends
4. WPM calculated

### Live Sample: [TypeTest Web](https://typetest-iota.vercel.app/)
